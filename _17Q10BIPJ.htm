Modificar datos de un empleado


Los datos de los empleados permanecen en la opción de Datos Generales, en cada una de las etiquetas o fichas que utilizamos para crearlos; de tal forma que si necesita modificar algún valor refierase a ellas para localizar el campo a modificar. Siga los siguientes pasos:
  1. Seleccione del Menú Principal las opciones Mantenimiento| Empleados | Datos Generales
  2. Seleccione el empleado al que necesita realizar alguna modificación,
  3. Ubique el campo a modificar entre las fichas Datos Principales, Forma de Pago, Datos Generales, Configuración o Ingresos Predeterminados,
  4. Escriba o seleccione los nuevos datos o valores y
  5. Al terminar, seleccione < Guardar >, para grabar los cambios.


En los datos generales hay algunos valores que para poder modificarse, se requiere estar utilizando el usuario SUPERVISOR, estos campos estan en la ficha de Configuración y son los siguientes: Porcentaje Máximo de los Descuentos, Dias sin contrato y Días Ausente por Licencia o Riesgos Profesionales del año actual.
