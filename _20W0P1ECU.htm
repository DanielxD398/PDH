Datos Generales

En la pantalla de datos generales encontramos cinco etiquetas en las cuales está la información particular de cada trabajador, veamos cada una de ellas:

  • Datos Principales
  • Forma de Pago
  • Datos Generales
  • Configuración
  • Ingresos Predeterminados


Los pasos para crear un empleado son los siguientes:

  1. Seleccione del Menú Principal las opciones Mantenimiento|Empleados|Datos Generales
  2. Seleccione <Nuevo> verá que se abren los campos para que pueda colocar la información
  3. Ingrese los datos del trabajador, en cada etiqueta
  4. Al terminar, seleccione <Guardar>
