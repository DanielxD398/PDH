Cambiar nombre a campos


Esta pantalla es útil para cambiar el nombre de los campos de otros ingresos y modificar así el nombre predefinido que PayDay tiene para los mismos. Los nombres pueden tener hasta 10 posiciones.

Para asignar un nuevo nombre a éstos campos, realice los siguiente:

  1. Seleccione del Menú Principal las opciones Mantenimiento| Configuración | Opciones
  2. Elija la etiqueta Cambiar Nombre a Campos,
  3. Ubique el campo que desea utilizar dentro de las categorías de Gravables y No Gravables,
  4. Verifique el Nombre Actual y coloque el nombre requerido en la columna Nombre Nuevo
  5. Al terminar, seleccione , para grabar los cambios realizados.

Nota: solo se le cambia el nombre que aparece, el comportamiento sigue siendo igual, ejemplo: si le cambia el nombre a OTR5 GAST por VIATICOS, a pesar que ahora se llama VIATICOS, el tratamiento en PayDay sigue siendo el de OTR5 GAST
