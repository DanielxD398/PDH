Crear nuevo Usuario


Tal como vimos anteriormente, para entrar a PayDay debemos teclear un código y un password con el cual se verifica si estamos autorizados a utilizar el programa. También vimos que tanto el código, como el password con el que se entra al programa es SUPERVISOR. Como estas claves son publicadas aquí y pueden ser conocidas por cualquier persona, es importante que la persona encargada de controlar la utilización del sistema, cree nuevos usuarios donde se especifique quienes son los autorizados a utilizarlo y les asigne las opciones que puedan accesar.

Crear un nuevo usuario
Para crear un nuevo usuario en PayDay:

· Seleccione del Menú Principal las opciones Otros | Administración | Usuarios.
· Luego tome la opción < Nuevo >.
· Llene los datos que se muestran en la siguiente tabla:

CAMPOS DESCRIPCION
Código Usuario Indique un código al nuevo usuario que va a poder utilizar el sistema de planilla. Este código puede ser el nombre de la persona a la que le pertenece. Si presiona " F2 " se mostrará la lista de los usuarios que han sido creados.
Nombre Se introduce el nombre del usuario, que representa el código colocado anteriormente. Si en el código colocó el nombre de la persona, aquí puede colocar el mismo.
Cambiar Password En esta opción aparece automáticamente para que el usuario coloque su contraseña para entrar, es decir su password. Este debe escribir la misma contraseña dos veces, en la línea Introduzca y Reintroduzca el password.


De esta forma cuando el usuario intente la próxima vez entrar a PayDay, deberá escribir su nuevo código y password. Posteriormente, el supervisor debe indicar las opciones que este nuevo usuario pueda accesar en el sistema, tal como se muestra en la siguiente figura:


Fig. No. 2.1 - Pantalla de Asignación de Acceso a Menús.

Para cambiar una opción, sólo dé un clic sobre la palabra SI y NO alternativamente y ellas cambiarán de forma automática. En esta pantalla de configuración de usuarios encontrará las siguientes opciones:

CAMPOS DESCRIPCIÓN
Nuevo Le permite crear un nuevo usuario dentro del sistema de planilla
Modificar Si desea hacer modificaciones a las opciones que tienen acceso los usuarios, utilice ésta opción
Grabar Con esta opción se guarda dentro del sistema las modificaciones que ha realizado en cuanto a las opciones a las que tiene acceso el nuevo usuario que está creando
Eliminar Se puede eliminar al (los) usuario(s) creados dentro de PayDay que tengan acceso al sistema
Cancelar Suspende cualquier modificación efectuada en esta pantalla
