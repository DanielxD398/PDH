Usuarios

Permite crear y eliminar usuarios o cambiarle sus derechos y contraseña (password). Esta pantalla debe ser de acceso restringido y en ella encontrará lo siguiente:

Código Usuario Indica el código del usuario del sistema de planilla. Si presiona "F2" aparecerá el listado de los usuarios que han sido creados.
Nombre Se introduce el nombre del usuario, que representa el código colocado anteriormente.
Cambiar Password Con esta opción puede cambiar el password de acceso del usuario que tenga seleccionado. Al tomar esta opción, se muestra una pantalla, solicitando el nuevo password. Coloque el mismo en los dos campos y luego seleccione la opción <Grabar>.
Asignar acceso a opciones del menú En el cuadro verde inferior están todas las opciones del menú para que elija a cuales va a dar acceso a cada usuario en el sistema.

Para realizar estos cambios, seleccione primero al usuario, luego tome la opción < Modificar > y autorice o niegue el acceso del usuario a cualquier menú o submenú del programa, solamente cambiando el estatus del acceso de Si a No y viceversa, dando < clic > sobre cada opción.

Observe que la lista de opciones tiene la forma del menú principal de PayDay y al momento de realizar cambios éstos se harán también en orden jerárquico. Por ejemplo, si desea colocar que un usuario no pueda cambiar salarios, sólo marque No en esa opción, ya que si lo hace en la opción Empleados, tampoco podrá crear o modificarlos empleados, ni entrar a Ajustes a los Acumulados.

Al terminar de efectuar los cambios no olvide seleccionar la opción < Grabar >.

Para eliminar a un usuario del sistema, selecciónelo y luego tome la opción < Eliminar >.

Cambiar Password Permite al usuario cambiar su propia contraseña o password para accesar al sistema, en la compañía en la que esté trabajando. Para hacerlo, sólo tome esta opción y teclee el nuevo password dos veces.


Cambiar Usuario Esta opción le permite cambiarse de usuario, en la sesión actual de trabajo.
