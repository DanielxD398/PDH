Copiar Archivos


Hacer un Respaldo de Archivos, es hacer una copia exacta de los datos de la empresa, para tenerlo como backup de información.

Este backup crea un archivo con extensión zip y lo guarda dentro de una carpeta llamada ¨Backup¨, localizada dentro del la carpeta donde están los datos de la empresa en PayDay, como por ejemplo:

Para hacer un respaldo, utilice cualquier opción del programa para Respaldo de Archivos y realice lo siguiente:
  • ( A ) Estando en el tab Copiar Archivos, como se muestra en la siguiente figura,
  • ( B ) Coloque un número a la copia que va a hacer. El número de copia es de sólo un caracter, por lo que puede usar los números del 0 al 9, luego puede reutilizarlos, seleccionando para ello el último de la lista, que estará ordenada cronológicamente. Es importante resaltar que con el transcurso del tiempo, cada copia tendrá cada vez más y más información, por lo que si se debe esporádicamente revisar el espacio en disco disponible.
    Para verificar la lista de las copias que tiene y/o reutilizarlas, presione en la lupa

  • (C) Coloque una Descripción o nombre a la copia que está realizando, el cual se recomienda que sea lo más descriptiva posible, de forma que cuando requiera utilizarla para reponer información, esté seguro de información está contenida en ella.
  • (D) Verifique la fecha y hora con que se grabará la copia,
  • (E) Escriba la contraseña para hacer copias. Esta es una contraseña pública y que tiene la finalidad que el usuario no se confunda con hacer copias o reponer una copia. La contraseña para hacer copias es NORMAL.
  • Por último presione para proceder con el backup o para no hacer la copia.
