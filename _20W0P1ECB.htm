Departamentos

PayDay le permite implementar la estructura organizacional de la empresa, clasificando a los empleados por departamentos, para luego listar los informes de acuerdo a esa clasificación.

Para crear un Departamento sólo haga lo siguiente:

  1. Seleccione Mantenimiento|Departamentos
  2. Presione < Nuevo >
  3. Indique el Código de Departamento: Será el código que identifique a cada departamento, debe tener un máximo de 6 caracteres. Ejemplos: VENTAS, ADMON, 01, 02.
  4. Coloque el Nombre del Departamento. Es el nombre que identificará a cada departamento. Ejemplo: VENTAS, ADMINISTRACION.
  5. El campo Identificación para la CSS, es útil para empresas con gran cantidad de empleados que requieran que sus fichas de la CSS vengan ordenadas por departamento. Para ello coloque en este campo, un código de dos posiciones, que no se repita por departamento.
  6. Al finalizar, presione < Guardar > y tendrá el departamento creado.
