Crear nuevo Usuario

Para crear un nuevo usuario en PayDay:

  • Seleccione la opción Otros|Administración|Usuarios.
  • Luego haga click en
  • Llene los datos que se muestran en la siguiente figura:

1. Código Usuario
Coloque un código al nuevo usuario que va a poder utilizar el sistema de planilla. Este código puede ser el nombre de la persona a la que le pertenece.

2. Nombre
Se introduce el nombre del usuario, que representa el código colocado anteriormente. Si en el código colocó el nombre de la persona, aquí puede colocar el mismo.

3. Cambiar Contraseña
Presione este botón y luego introduzca y reintroduzca la contraseña del usuario que está creando. De esta forma cuando el usuario intente la próxima vez entrar a PayDay, deberá escribir su nuevo código y password.

4. Usuario de Solo Leer
Este parámetro no permitirá al usuario hacer cambios en los datos de la empresa. Solo podrá entrar en las opciones para leer o revisar información -sin permitírsele hacer ningún tipo de modificaciones.

5. NO puede reponer Backups
Esta opción restringe al usuario la opción de Reponer un Backup.

6. Modificar Ingresos Predeterminados
Permite que el usuario pueda modificar los valores en la pantalla de Ingreos Predeterminados directamente. Si no está habilitada esta opción, el usuario debe hacer los cambios por la pantalla de Cambio de Salario.

7. NO puede Eliminar Backup
Restringe al usuario la posibilidad de eliminar respaldos de archivos.

8. Roles Asignados
Estos son los roles que se pueden asignar a cada usuario. Dé click en Ok para asignar ese rol al usuario nuevo.

9. Contraseña de Documentos
Es la contraseña (individual, por usuario) con la que se encriptarán los archivos en formato PDF que envíe por correo. Recomendamos que no sea la misma contraseña que utiliza para acceder a la compañía.

Nota: Para utilizar esta característica, debe activar la opción "Encriptar y colocar contraseña a los informes en PDF" en el menú Mantenimiento|Configuración|Opciones, pestaña Seguridad.
