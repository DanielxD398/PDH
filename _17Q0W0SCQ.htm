OJO: Incluir ....Para entrar a PayDay

*************
Tal como vimos anteriormente, para entrar a PayDay debemos teclear un código y un password con el cual se verifica si estamos autorizados a utilizar el programa. También vimos que tanto el código, como el password con el que se entra al programa es SUPERVISOR. Como estas claves son publicadas aquí y pueden ser conocidas por cualquier persona, es importante que la persona encargada de controlar la utilización del sistema, cree nuevos usuarios donde se especifique quienes son los autorizados a utilizarlo y les asigne las opciones que puedan accesar.

Cambiar Password

Usted puede cambiar su propia contraseña (password) para accesar al sistema en la compañía en la cual está trabajando.

Para hacerlo, seleccione la opción Otros | Administración | Cambiar Password y teclee su nueva contraseña en las opciones El Password nuevo es y Reintroduzca el Password.

El primer paso para comenzar a trabajar con PayDay es ingresar la información necesaria para confeccionar las planillas.
