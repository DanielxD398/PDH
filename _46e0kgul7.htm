Envío otros archivos

En algunas ocasiones, no es necesario enviar el backup completo al departamento de soporte de PayDay; cuando se le pide uno o algunos archivos específicos, usted puede generar un archivo comprimido que los contenga y luego enviar este archivo via FPT.

Con esta opción podemos enviarlo fácilmente, solamente ubique el archivo seleccionando la Ruta / ubicación y luego tome la opción ZIP - via FTP.
Por seguridad el archivo debe ser tipo "zip, rar o 7z", es decir ya debe estar comprimido.
